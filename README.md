# AI Vision App

AI Vision App is a full-stack web application where users can upload images, analyze them using AI, and generate new images from text prompts.

This project is built to learn how modern AI-powered applications work end-to-end, including frontend, backend, databases, cloud storage, and AI model integration.

## Features
- Upload images (JPEG, PNG)
- Image analysis (captioning, VQA, OCR)
- Text-to-image generation
- Image variations

## Tech Stack
- Frontend: Next.js (TypeScript, Tailwind CSS)
- Backend: FastAPI (Python)
- Database: PostgreSQL (Supabase)
- Storage: AWS S3
- AI: Vision-Language Models & Diffusion Models
